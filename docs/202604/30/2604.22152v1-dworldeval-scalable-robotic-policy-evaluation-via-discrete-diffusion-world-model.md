## 摘要
在成千上万个环境和任务中评估机器人策略，对于现有方法而言是不可行的。这促使我们需要一种新的方法论来进行可扩展的机器人策略评估。在本文中，我们提出了 dWorldEval，它使用离散扩散世界模型作为机器人策略的可扩展评估代理。具体而言，dWorldEval 将包括视觉、语言和机器人动作在内的所有模态映射到一个统一的 token 空间，并通过单个基于 Transformer 的去噪网络对其进行建模。基于该架构，我们采用了稀疏关键帧记忆来维持时空一致性。我们还引入了一个表示任务完成程度的进度 token。在推理时，该模型联合预测未来的观测结果和进度 token，从而在进度达到 1 时自动判定任务成功。大量实验表明，dWorldEval 在 LIBERO、RoboTwin 以及多个真实机器人任务上显著优于以往的方法（如 WorldEval、Ctrl-World 和 WorldGym）。它为构建用于大规模机器人评估的世界模拟器开辟了一种新的架构范式。

## Abstract
Evaluating robotics policies across thousands of environments and thousands of tasks is infeasible with existing approaches. This motivates the need for a new methodology for scalable robotics policy evaluation. In this paper, we propose dWorldEval, which uses a discrete diffusion world model as a scalable evaluation proxy for robotics policies. Specifically, dWorldEval maps all modalities - including vision, language, and robotic actions - into a unified token space, modeling them via a single transformer-based denoising network. In this paper, we propose dWorldEval, using a discrete diffusion world model as a scalable evaluation proxy for robotics policy. Specifically, it maps all modalities, including vision, language, and robotics action into a unified token space, then denoises them with a single transformer network. Building on this architecture, we employ a sparse keyframe memory to maintain spatiotemporal consistency. We also introduce a progress token that indicates the degree of task completion. At inference, the model jointly predicts future observations and progress token, allowing automatically determine success when the progress reaches 1. Extensive experiments demonstrate that dWorldEval significantly outperforms previous approaches, i.e., WorldEval, Ctrl-World, and WorldGym, on LIBERO, RoboTwin, and multiple real-robot tasks. It paves the way for a new architectural paradigm in building world simulators for robotics evaluation at scale.

#### 1. 核心问题与整体含义（研究动机和背景）
随着通用机器人操纵策略（如 VLA 模型）的快速发展，如何在成千上万种环境和任务中高效、低成本地评估这些策略成为了重大挑战。传统的实机评估成本极高，而基于物理引擎的仿真评估则面临资产建模繁重的问题。
现有的生成式世界模型虽被视为一种潜在的评估代理，但存在两大核心缺陷：
*   **动作控制力差**：模型往往忽略错误的动作输入，产生“成功幻觉”（即无论动作对错，视频都显示成功）。
*   **时空不一致性**：在长程预测中，物体容易发生形变、消失或位置漂移。
本文提出的 **dWorldEval** 旨在通过一种全新的离散扩散架构，构建一个高可靠、可控且能自动评分的世界模型，作为机器人策略评估的扩展代理。

#### 2. 论文提出的方法论
dWorldEval 抛弃了传统的视频扩散架构（将动作作为辅助条件注入），转而采用**掩码离散扩散（Masked Discrete Diffusion, MDD）**。
*   **统一 Token 空间**：使用 MAGVIT-v2 编码视觉、LLaDA 编码语言、FAST 编码动作块，将所有模态映射到统一的离散 Token 序列中。通过 Transformer 的自注意力机制，使视觉 Token 直接对等地位地关注动作 Token，显著增强了动作的可控性。
*   **稀疏关键帧记忆（Sparse Keyframe Memory）**：引入历史关键帧作为上下文锚点，通过滑动窗口采样最近的 $K$ 帧低分辨率图像，有效缓解了长程预测中的时空漂移。
*   **进度 Token（Progress Token）**：在 Token 序列中加入一个表示任务完成度（0 到 1）的离散 Token。模型在预测未来视觉状态的同时联合预测该进度值，当进度达到 1 时自动判定为任务成功。
*   **联合去噪目标**：通过最小化掩码 Token 的重构损失进行训练，推理时采用迭代并行解码（Iterative Parallel Decoding）同时生成图像和进度分值。

*   **对比基准（Baselines）**：对比了当前主流的视频扩散评估模型，包括 **WorldEval**、**WorldGym** 和 **Ctrl-World**。
*   **评估指标**：除了标准的 LPIPS，本文创新性地提出了 **$\Delta$-LPIPS**（测量状态转换的感知保真度，而非绝对状态），以及 Pearson 相关系数 $r$ 和排名违规率（MMRV）来衡量评估模型与真实执行结果的一致性。

*   **硬件与时长**：使用 **8 张 NVIDIA H800 GPU** 进行训练，训练时长为 15 个 epoch。
*   **推理效率**：在单张 H800 上，评估一条完整轨迹约需 30-90 秒（约 1.5 秒/帧）。

#### 5. 实验数量与充分性
论文进行了系统且广泛的实验：
*   **跨平台验证**：涵盖了两个大型仿真环境和一个真实机器人平台。
*   **异构策略评估**：评估了不同架构（如 Diffusion Policy, DexVLA）和不同训练阶段的策略。
*   **消融实验**：详细验证了“稀疏记忆模块”对一致性的贡献，以及“进度 Token”作为自动评分器的准确性。
*   **动作敏感性测试**：通过随机打乱动作序列（Action Shuffling）验证了模型确实是受动作驱动而非单纯的视觉外推。

#### 6. 主要结论与发现
*   **动作可控性显著提升**：dWorldEval 在处理失败或分布外（OOD）动作时，能忠实地渲染出失败结果，而非幻觉出成功。
*   **高相关性**：模型估计的成功率与真实执行成功率表现出极强的线性相关性（Pearson $r \approx 0.9$），能够准确对不同机器人策略进行排名。
*   **时空稳定性**：稀疏关键帧记忆成功抑制了长程预测中的物体漂移，使得 20 步以上的长程评估成为可能。

#### 7. 优点
*   **架构创新**：将动作视为“一等公民”Token，从根本上解决了视频扩散模型中动作信号被视觉先验覆盖的问题。
*   **自动闭环评估**：内置进度 Token 实现了无需外部奖励函数或人工干预的自动评分。
*   **新指标 $\Delta$-LPIPS**：为衡量世界模型的动作敏感性提供了一个更科学的诊断工具。
*   **多视图一致性**：支持同步生成第三方视角和腕部相机视角，符合真实机器人部署需求。

#### 8. 不足与局限
*   **推理延迟**：虽然比实机快，但 1.5 秒/帧的生成速度对于超大规模（如百万级）的策略筛选仍有加速空间。
*   **任务范畴**：目前主要集中在桌面操纵任务，对于更复杂的全身运动或大范围导航场景的泛化能力尚待验证。
*   **数据依赖**：模型性能仍高度依赖于训练数据中失败样本的质量和多样性。
