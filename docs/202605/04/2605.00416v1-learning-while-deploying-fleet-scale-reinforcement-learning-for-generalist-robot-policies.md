## 摘要
通用机器人策略日益受益于大规模预训练，但仅靠离线数据不足以实现稳健的现实世界部署。部署后的机器人会遇到分布偏移、长尾失败、任务变化以及人类纠错机会，而固定的演示数据集无法完全捕捉这些情况。我们提出了“边部署边学习”（Learning While Deploying, LWD），这是一个车队级的离线到在线强化学习框架，用于通用视觉-语言-动作（VLA）策略的持续后训练。LWD 从预训练的 VLA 策略开始，通过利用在机器人车队中收集的自主运行和人类干预，闭合了部署、共享物理经验、策略改进和重新部署之间的循环。为了稳定从异构、稀疏奖励的车队数据中学习的过程，LWD 将用于稳健价值估计的分布隐式价值学习（DIVL）与通过伴随匹配进行 Q 学习（QAM）相结合，用于流式 VLA 动作生成器中的策略提取。我们在由 16 台双臂机器人组成的车队上验证了 LWD，涵盖了八项现实世界的操纵任务，包括语义杂货补货和 3-5 分钟的长程任务。随着车队经验的积累，单一通用策略不断改进，平均成功率达到 95%，其中在长程任务上的提升最为显著。

## Abstract
Generalist robot policies increasingly benefit from large-scale pretraining, but offline data alone is insufficient for robust real-world deployment. Deployed robots encounter distribution shifts, long-tail failures, task variations, and human correction opportunities that fixed demonstration datasets cannot fully capture. We present Learning While Deploying (LWD), a fleet-scale offline-to-online reinforcement learning framework for continual post-training of generalist Vision-Language-Action (VLA) policies. Starting from a pretrained VLA policy, LWD closes the loop between deployment, shared physical experience, policy improvement, and redeployment by using autonomous rollouts and human interventions collected across a robot fleet. To stabilize learning from heterogeneous, sparse-reward fleet data, LWD combines Distributional Implicit Value Learning (DIVL) for robust value estimation with Q-learning via Adjoint Matching (QAM) for policy extraction in flow-based VLA action generators. We validate LWD on a fleet of 16 dual-arm robots across eight real-world manipulation tasks, including semantic grocery restocking and 3--5 minute long-horizon tasks. A single generalist policy improves as fleet experience accumulates, reaching an average success rate of 95%, with the largest gains on long-horizon tasks.

### 1. 核心问题与整体含义
*   **研究动机**：尽管视觉-语言-动作（VLA）模型在离线预训练中表现出色，但在真实部署中仍面临分布偏移（Distribution Shift）、长尾失败和环境多样性等挑战。
*   **核心问题**：如何利用机器人机群（Fleet）在部署过程中产生的真实交互数据（包括成功、失败、部分进展和人类干预），实现通用策略的持续进化，而非仅仅将其视为静态的测试过程。
*   **整体含义**：提出了 **LWD (Learning While Deploying)** 框架，将部署过程转化为数据飞轮，通过“离线预训练 + 在线持续强化学习”闭环，提升通用机器人的鲁棒性和长程任务处理能力。

### 2. 方法论
LWD 结合了分布价值学习与高效的策略提取技术，主要包含两个关键算法组件：
*   **分布隐式价值学习 (DIVL)**：
    *   **核心思想**：改进了隐式 Q 学习（IQL）。由于机群数据具有高度异构性和多模态性（同一状态下可能有多种结果），DIVL 不再估计标量期望值，而是学习回报的**分类分布（Categorical Distribution）**。
    *   **技术细节**：通过最小化负对数似然来拟合分布，并使用该分布的特定分位数（Quantile）作为 TD 目标的引导值。
    *   **自适应策略**：引入基于分布熵的自适应 $\tau$ 参数，在不确定性高时保持保守，在信心足时追求乐观。
*   **伴随匹配 Q 学习 (QAM) 进行策略提取**：
    *   **核心思想**：针对基于流匹配（Flow Matching）的 VLA 策略，直接通过 ODE 求解器进行反向传播既不稳定又耗时。
    *   **技术细节**：QAM 将 Critic 提供的动作梯度转化为流模型每一步的局部回归目标，从而在不破坏生成模型表达能力的前提下，稳定地将价值提升转化为策略改进。
*   **训练流程**：分为离线预训练（利用演示、历史运行和人类引导的失败探索数据）和在线持续训练（机群自主运行 + 混合离线数据回放）。

*   **对比基准 (Baselines)**：

    *   **RECAP**：一种基于优势加权的离线后训练方法。

### 6. 主要结论与发现
*   **性能提升**：LWD (Online) 达到了 **95% 的平均成功率**，显著优于 SFT (76%) 和其他基线。
*   **长程任务优势**：在长程任务中，RL 的优势最明显（得分 0.91 vs SFT 0.68），因为它能通过 TD 误差传播奖励信号，并“缝合”部分成功的经验。

*   **分布价值的重要性**：消融实验证明，在异构机群数据下，分布价值估计比标量估计在长程任务上提升了 16.7%。

### 7. 优点
*   **系统性创新**：成功闭环了“部署-收集-训练-再部署”的工业级数据飞轮。
*   **算法适配性**：DIVL 解决了机群数据的多模态问题，QAM 解决了大模型流策略的训练稳定性问题。
*   **真实感强**：完全基于真实物理机器人实验，任务复杂度高（如功夫茶），具有极强的实用参考价值。

### 8. 不足与局限
*   **安全建模缺失**：目前的框架未显式加入执行安全约束，在更敏感的环境中部署可能存在风险。
*   **语言理解深度**：长程任务仍依赖相对简单的指令，对于需要复杂逻辑推理或多轮对话的任务支持有限。
*   **更新频率**：虽然实现了在线学习，但策略更新仍是基于固定周期的异步同步，而非完全实时的流式更新。
*   **算力成本**：VLA 模型规模较大，持续后训练对云端算力和网络带宽（模型分发）有较高要求。
